Thalariond

Written in Ruby and Ruby on Rails, active development

Thalariond is a suite of services providing a centralized user management system, which gives the system administrator finegrained control over users and permissions. It consists of a database and a Ruby on Rails applications allowing access to users, permissions, domains and email addresses, as well as an LDAP backend written in plain Ruby. LDAP is the de facto standard for user directories in open source projects as well as enterprise solutions.

Headbanger is a music recommendation and discovery platform. It uses hybrid collaborative and content-based filtering to determine novel artists and albums based on the user's previous ratings. Fuzzy Car Controller

Written in Java

Fuzzy logic was introduced by Lotfi Zadeh in 1965. He proposed a new approach to causative computing, drawing the analogy with humans which usually reason in “fuzzy” terms.
This project implements a fuzzy rule-based system using Zadeh's fuzzy logic, including a practical application – the fuzzy car controller. It parses sensorial data and predefined rules to allow a race-car to navigate a simplified race track autonomously.
